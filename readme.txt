Тестовое задание выполнено на технологиях .Net Core 3.1 на бэкенде и Vue JS 2.6.11 на фронтенде.

При запуске сервера будет автоматически приаттачена база SQL с изначально созданной структурой таблиц, а также занесены тестовые данные.

Для входа в приложение можно использовать сочетание логин/пароль - admin/admin , т.к. эта учётка также создаётся при инициализации.
